# Calculate geometric mean

Calculate geometric mean

## Usage

``` r
geometric_mean(x)
```

## Arguments

- x:

  Numeric vector of values for which to calculate the geometric mean.

## Value

The geometric mean of the values in `x`. `NA` values are ignored.

## Details

Used in
